## MES的定义及作用

MES(Manufacturing Execution System)即制造企业生产过程执行系统，是一套面向制造企业车间执行层的生产信息化管理系统，是一个用来跟**踪生产进度**，**库存情况**，**工作进度**和其它**进出车间的操作管理**相关的信息流。

MES是介于ERP(企业资源计划系统)和自控系统（DCS、PCL等）之间的系统，是管控一休化的桥梁，对于已经实现ERP系统的企业来说，缺少MES系统就相当于在计划与过程控制之间形成了“断层”。MES属于与生产过程连接的企业信息系统，是实现企业综合自动化的关键环节。

生产执行系统通过控制包括物料、设备、人员、流程指令和设施在内的所有工厂资源来提高制造竞争力，确保整个生产行为的最优。生产信息管理系统主要作用是、一方面实时收集和传递生产数据，通过设计的功能提供给所有需要的管理者，使他们迅速的知晓生产变化，对生产进行有针对性的调控，实现对生产有效的管控；另一方面，通过对生产数据的分析，找出规律，改进生产组织和管理的方法，优化生产管理的体系。
